🚨 SPAM X - Smart Emergency Response System

📌 Overview

SPAM X (Smart Emergency Response System) is a web-based emergency assistance platform developed to provide quick and reliable emergency support during critical situations.

The system helps users instantly access:

- 🚑 Ambulance Services
- 🏥 Nearby Hospitals
- 🩸 Blood Donors
- 🚓 Police Assistance
- 🚒 Fire Stations
- 🚨 SOS Emergency Alerts
- 🛣 Highway Patrol Support

The main objective of SPAM X is to reduce emergency response time and improve public safety using smart digital solutions.

---

✨ Features

🚑 Emergency Contacts

- Quick access to police, ambulance, fire station, and highway patrol services.

🏥 Nearby Hospitals

- Displays nearby hospitals and emergency medical support.

🩸 Blood Donor Support

- Helps users find available blood donors quickly.

📍 Live Location Tracking

- Supports real-time location assistance during emergencies.

🚨 SOS Alert System

- Emergency SOS button for immediate help alerts.

🔐 Login & Registration

- Secure user login and registration system.

▶️ How to Run the Project

Step 1

Clone the repository

git clone https://github.com/YOUR_USERNAME/Smart-Emergency-Response-System.git

Step 2

Move into backend folder

cd backend

Step 3

Install Flask

pip install flask

Step 4

Run the application

python app.py

Step 5

Open browser

http://127.0.0.1:5000
